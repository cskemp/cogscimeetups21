San Francisco 2021

Event Details

Join us for a casual social event in the heart of San Francisco as part of the Cognitive Science Society Conference 2021! We welcome researchers at all levels from across the Bay Area. The event will be held in a private backyard in Duboce Triangle on Tuesday, July 27th from 5-9pm PST. Please register below to attend and receive specific location details.

We are following the Local Meetup model to bring local researchers together for in-person gatherings as part of the virtual Cognitive Science Society Conference. We are following local COVID-19 regulations, which currently allow gatherings. The event will be held outdoors, but access requires a brief indoor passage and restrooms are only available indoors. Masks are optional and vaccination is recommended. Participants must follow the code of conduct described here. The event will have a maximum capacity of 40 attendees.

Food and Drink

We will have some basic refreshments available but please BYOB.

The event is free and registration for the Cognitive Science Society Conference 2021 is not required. However, please register for this event using the form below.
